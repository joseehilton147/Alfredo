# Alfredo AI Product Overview

Alfredo AI is an AI-powered video transcription and analysis tool that uses OpenAI Whisper for automatic audio transcription.

## Core Features
- Transcribe audio from videos automatically using OpenAI Whisper
- Process local video files
- Download and process YouTube videos
- Save transcriptions in JSON format
- Batch processing of multiple videos
- Multi-language support (default: Portuguese)

## Target Use Cases
- Content creators needing video transcriptions
- Researchers analyzing video content
- Accessibility improvements for video content
- Content analysis and documentation

## Key Value Propositions
- Simple CLI interface for easy automation
- Support for multiple video formats (mp4, avi, mkv, mov, wmv, flv, webm)
- Docker containerization for easy deployment
- Clean architecture for maintainability and extensibility

## User Interface
- Command-line interface (CLI) as primary interaction method
- Docker support for containerized execution
- Makefile for common development tasks
